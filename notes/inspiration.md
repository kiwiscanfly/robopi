# Inspiration

A collection of YouTube videos to watch for robotics inspiration and learning.

---

## [Can I build an AI Robot Arm to do my chores?](https://youtu.be/2hYSKIdKS9w)
**Channel:** Back to Engineering (iulia)

Building a 3D printed robot arm using Hugging Face's LeRobot framework to train visual-language-action models (VLAs) for tasks like folding laundry and making coffee. Covers teleoperation setup, motor calibration, and dataset generation for policy training with the SO-ARM100.

## [Using LLMs to write microcontroller code (AI runs robot arm)](https://youtu.be/aqYN7jCgId0)
**Channel:** Back to Engineering (iulia)

Using GitHub Copilot in VS Code to write servo control code for a robot arm — smoothing twitchy movements, adding angle control, and building functions to make the arm wave and nod. Good example of AI-assisted embedded code iteration.

## [I build the simplest and cheapest Robot Arm at home (3D printed)](https://youtu.be/xILy-UI7aAQ)
**Channel:** Back to Engineering (iulia)

Building the open-source Grippy Bot from 3D printed parts and cheap servo motors, controlled via Raspberry Pi Pico or Arduino. Covers assembly pitfalls, twitching fixes, potentiometer removal, and Python vs C++ code options. Good beginner arm project before moving to LeRobot.

## [From 0 to building a robot arm with Arduino](https://youtu.be/c6mrcNEFBoA)
**Channel:** Back to Engineering (iulia)

Series starter: absolute beginner introduction to Arduino, breadboards, servos, and potentiometers, building up to a first robot arm prototype. Good entry point before moving to more capable microcontrollers or LeRobot.

## [I tried to learn ROS in 8 hours (and set a robot on fire)](https://youtu.be/MTQa9OmIPvE)
**Channel:** Back to Engineering (iulia)

Installing ROS 2 Jazzy on Linux Mint and a Raspberry Pi 5, learning nodes/topics/services via turtlesim, then wiring up real servos and LEDs on the Pi and controlling them with teleop arrow keys over a shared ROS_DOMAIN_ID. Very relevant — same stack as this project's stage 3.

## [The Beginner's Electronics Roadmap (Projects Included)](https://youtu.be/lsvbCYLFZ84)
**Channel:** Back to Engineering (iulia)

Practical roadmap for getting into microcontrollers — choosing between Arduino and Raspberry Pi Pico, kit vs components, Python vs C++, IDE setup (VS Code, Thonny), and how to find and combine project ideas. Includes blink → sensors → servo → robot arm progression.

## [The Physical AI Roadmap: From Microcontrollers to Autonomous Robots](https://youtu.be/5Cn_KCHVEEM)
**Channel:** Back to Engineering (iulia)

Big-picture roadmap from LED blink to autonomous robots: microcontrollers → ROS 2 → simulation → computer vision → VLA models. Aimed at ML/data science people moving into physical AI. Mirrors the arc of this project.

## [Robot arm ultimate guide: What I learned from 5 builds](https://youtu.be/nTJsVvWFUVE)
**Channel:** Back to Engineering (iulia)

Lessons from five different robot arm builds — comparing servo vs stepper motors, 3D printed designs (Grippy Bot, EEZYBoy MK3, SO-ARM100), Arduino vs ESP32 vs Pico, calibration tips, wiring, and code abstraction. Ends with a tease of ROS integration.
